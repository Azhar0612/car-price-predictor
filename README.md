# 🚗 Car Price Predictor

## 📌 Overview
This project predicts the price of used cars using Machine Learning techniques.  
It is built using Python and trained on a real-world dataset.

---

## ⚙️ Technologies Used
- Python 🐍
- Pandas
- NumPy
- Scikit-learn

---

## 📊 Dataset
- Dataset used: Quikr Car Dataset (`quikr_car.csv`)
- Contains details like:
  - Car Name
  - Year
  - Fuel Type
  - Price

---

## 🚀 Features
- Predicts car price based on input data
- Data preprocessing and cleaning
- Machine Learning model training

---

## ▶️ How to Run

```bash
pip install pandas numpy scikit-learn
python car_price.py
